User Guide

Welcome to TaskFlow, your personal task management tool!

Features:

  • Add, edit, and delete tasks.
  • Organize tasks into groups.
  • Set priority levels for tasks.
  • Mark tasks as complete.
  • Export and import tasks in JSON format.
  • Replace existing tasks with imported tasks.
  • Delete all tasks at once.
  • Tasks are saved locally on your browser.

How to Use:

Use the form to add tasks, specify group names to categorize, and select priority levels.

Click on the edit (‚úèÔ∏è) or delete (üóëÔ∏è) icons to modify tasks.

Use the hamburger menu to navigate between the Task List, User Guide, and manage tasks.

Exporting Tasks:

Select "Export Tasks" from the hamburger menu to copy your tasks to the clipboard in JSON format.

Importing Tasks:

Copy a JSON-formatted task list to your clipboard, then select "Import Tasks" to add them to your existing tasks.

Replacing Tasks:

Copy a JSON-formatted task list to your clipboard, then select "Replace Tasks" to replace your current tasks.

Deleting All Tasks:

Select "Delete All Tasks" to remove all tasks from your list.

Viewing the AI Prompt:

Select "Prompt" from the hamburger menu to view the original AI prompt used to build this application.
